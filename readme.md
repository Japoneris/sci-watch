# Scientific watch tracking interface

This app let you monitor:

- HackerNews topics
- Arxiv Papers




# Env file

For arxiv, configure env file with the following fields:

```sh
OPENAI_API_VERSION=2024-12-01-preview
APIM_ENDPOINT=https://apiemq.ingenico.com
APIM_API_KEY=
AZURE_TENANT_ID=
AZURE_CLIENT_ID=
AZURE_CLIENT_SECRET=
```
